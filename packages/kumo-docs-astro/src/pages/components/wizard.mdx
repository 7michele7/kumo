## Anatomy

The Wizard is a compound component. Compose only the pieces you need:

```
Wizard.Fullscreen — portal overlay
  └── Wizard.Grid (optional — decorative wireframe)
        └── Wizard — step state, context, and navigation
              └── Wizard.Steps — filters and indexes active steps
                    └── Wizard.Step — individual step with card-stack animation
                          └── Wizard.Page — card surface with title, body, and footer
```

</ComponentSection>

## Usage

```tsx
import { useState } from "react";
import { Wizard, Button } from "@cloudflare/kumo";

export default function Example() {
  const [step, setStep] = useState(0);

  return (
    <Wizard.Fullscreen open onClose={() => {}}>
      <Wizard step={step} onStepChange={setStep}>
        <Wizard.Steps>
          <Wizard.Step stepKey="first" label="First">
            <Wizard.Page
              title="First step"
              footer={
                <>
                  <div />
                  <Button onClick={() => setStep(1)}>Next</Button>
                </>
              }
            >
              <p>Step content here.</p>
            </Wizard.Page>
          </Wizard.Step>
          <Wizard.Step stepKey="second" label="Second">
            <Wizard.Page
              title="Second step"
              footer={
                <>
                  <Button variant="secondary" onClick={() => setStep(0)}>
                    Back
                  </Button>
                  <Button>Done</Button>
                </>
              }
            >
              <p>More content here.</p>
            </Wizard.Page>
          </Wizard.Step>
        </Wizard.Steps>
      </Wizard>
    </Wizard.Fullscreen>
  );
}
```

### `useWizardGrid()`

<p>
  Bridges `Wizard` and `Wizard.Grid` by tracking the active card's height and
  managing transition state. Returns `gridProps` to spread onto `Wizard.Grid`.
</p>

```tsx
import { useWizardGrid } from "@cloudflare/kumo";

const { gridProps, onActiveStepElementChange } = useWizardGrid();

<Wizard.Grid {...gridProps}>
  <Wizard step={step} onStepChange={setStep}>
    <Wizard.Steps>...</Wizard.Steps>
  </Wizard>
</Wizard.Grid>;
```
